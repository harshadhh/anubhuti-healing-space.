Quick Guide

Use this admin panel to manage all images and content on the Anubhuti Healing Space website. All images are uploaded directly to Cloudinary and served from there across the website.

How it works: Select an image, add a label/name, and click Upload. The image goes to Cloudinary and the URL is saved. Copy the URL and paste it into the relevant HTML file to make it live on the website.
Cloudinary Preset: Set your upload_preset and cloud_name in Cloudinary Settings before uploading.
